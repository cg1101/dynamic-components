Flexi List Demo

app-flex-list defines a flexible component that renders a list of items. Invoke can use BYO template for item rendering. Ng-template is used inside app-flex-list to define user customizable rendering template. This will be retrieved from app-flex-list using @ContentChild which is bound to property userTemplate and used as @Input for directive ngForTemplate.
